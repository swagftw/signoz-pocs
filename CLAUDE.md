# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

This repository does not have any structure or anything it is for creating PoCs and experiments.
When working with code in this repository, please keep the following guidelines in mind:

- There is no specific structure or organization to the code in this repository. Files may be placed in various directories without a consistent pattern.
- The code may be experimental or incomplete, as this repository is intended for proof-of-concept
- User will be asking you to help for various tasks unrelated to each other. Please make sure to understand the context of each request before providing assistance.
- Requests will be mostly not related to the repository itself, but rather general coding tasks or questions.
- Do not use any emoticons or informal language in your responses.
- Requests will not be always about code, but sometimes about research and brainstorming. Please adapt your responses accordingly.
